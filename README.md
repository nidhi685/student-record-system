# Student Record System

A MERN Stack based Student Record System for managing student information, attendance, marks, and subjects.

## Features

### Admin
- Manage Students
- Manage Attendance
- Manage Marks
- Manage Subjects

### Student
- View Profile
- View Attendance
- View Marks
- View Subjects

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB

## Installation

```bash
npm install
```

## Run Project

### Frontend

```bash
npm start
```

### Backend

```bash
npm run dev
```
